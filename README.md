# TPI-OrganizacionEmpresarial-ChatBotVacaciones
Chatbot de Solicitud de Vacaciones
Descripción

Proyecto desarrollado para la materia Organización Empresarial de la Tecnicatura Universitaria en Programación a Distancia.

El sistema simula un chatbot para la gestión de solicitudes de vacaciones. Permite consultar el saldo disponible de días, validar la cantidad solicitada y actualizar automáticamente la información almacenada en un archivo CSV.

Tecnologías utilizadas
Python
CSV
GitHub
Archivos del proyecto
main.py
empleados.csv
README.md
Informe.pdf
Funcionamiento
Ejecutar el archivo main.py.
Ingresar el DNI del empleado.
Consultar los días disponibles.
Ingresar la cantidad de días solicitados.
El sistema aprobará o rechazará la solicitud.
En caso de aprobación, se actualizará el archivo empleados.csv.
